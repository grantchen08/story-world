# Asset Specification: BGM - Royal Court

## 1. Description
Background music for the Palace/Throne Room.

## 2. Technical Requirements
*   **File Name**: `bgm_royal_court.mp3`
*   **Loop**: Yes.

*   **Instrumentation**: Brass (ancient horns), bells, formal rhythm.
